# Data Engineer — Job Description

## Role Overview
We are looking for a **Data Engineer** to build and maintain data pipelines, warehouses, and analytics infrastructure. You will ensure reliable data flow from source systems to analytics and ML platforms.

## Responsibilities
- Design and build scalable ETL/ELT data pipelines
- Maintain data warehouses and data lakes (Snowflake, BigQuery, Redshift)
- Implement data quality checks and monitoring
- Build streaming data pipelines using Kafka, Spark Streaming, or Flink
- Collaborate with data scientists and analysts to provide clean, reliable datasets
- Optimize query performance and data storage costs
- Maintain documentation for data models and pipeline architecture

## Required Skills
- **Languages**: Proficiency in Python and SQL
- **Pipelines**: Experience with Apache Airflow, dbt, or similar orchestration tools
- **Cloud**: Hands-on experience with AWS (S3, Glue, Redshift) or GCP (BigQuery, Dataflow)
- **Databases**: Strong SQL skills; experience with both OLTP and OLAP systems
- **Big Data**: Familiarity with Spark, Hadoop ecosystem
- **Version Control**: Git proficiency

## Preferred Skills
- Experience with real-time streaming (Kafka, Flink)
- Knowledge of data governance and cataloging tools
- Familiarity with ML pipelines and feature stores
- Experience with containerization and infrastructure-as-code
- Knowledge of data privacy regulations (GDPR, CCPA)

## Minimum Qualifications
- Bachelor's degree in Computer Science, Data Science, or related field
- 2+ years of professional data engineering experience
- Strong analytical and problem-solving skills

## Compensation
- Competitive salary based on experience
- Health insurance, flexible hours, remote-friendly
