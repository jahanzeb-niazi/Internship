# Backend Engineer — Job Description

## Role Overview
We are looking for a **Backend Engineer** to design, build, and maintain scalable server-side systems. You will work closely with frontend engineers, data teams, and DevOps to deliver reliable APIs and services.

## Responsibilities
- Design and implement RESTful APIs and microservices
- Build and maintain backend systems using Node.js, Python, or Go
- Work with relational (PostgreSQL) and NoSQL (MongoDB, Redis) databases
- Deploy and manage services on cloud platforms (AWS, GCP, or Azure)
- Write unit and integration tests; maintain CI/CD pipelines
- Collaborate on system architecture and technical design decisions
- Monitor application performance and troubleshoot production issues

## Required Skills
- **Languages**: Proficiency in at least one of: Node.js, Python, Go, Java
- **APIs**: Experience building REST APIs; familiarity with GraphQL is a plus
- **Databases**: Hands-on experience with SQL and at least one NoSQL database
- **Cloud**: Experience with AWS (EC2, S3, Lambda) or equivalent cloud services
- **Version Control**: Proficiency with Git and collaborative development workflows
- **Testing**: Experience with unit testing frameworks and CI/CD tools

## Preferred Skills
- Experience with containerization (Docker, Kubernetes)
- Familiarity with message queues (RabbitMQ, Kafka)
- Knowledge of AI/ML frameworks (LangChain, TensorFlow) is a strong plus
- Experience with infrastructure-as-code (Terraform, CloudFormation)

## Minimum Qualifications
- Bachelor's degree in Computer Science, Software Engineering, or related field
- 2+ years of professional backend development experience
- Strong problem-solving skills and ability to work independently

## Compensation
- Competitive salary based on experience
- Health insurance, flexible hours, remote-friendly
