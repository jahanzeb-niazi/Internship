# Engineering Rubric — Candidate Scoring Guide

## Overview
This rubric defines how candidates are scored across key dimensions. Each dimension is rated on a 1–5 scale. The weighted total determines the qualification decision.

## Scoring Dimensions

### 1. Technical Skills (Weight: 35%)
| Score | Description |
|-------|-------------|
| 5 | Expert-level proficiency in all required skills + strong preferred skills |
| 4 | Proficient in all required skills + some preferred skills |
| 3 | Meets most required skills, missing 1–2 areas |
| 2 | Meets some required skills but has significant gaps |
| 1 | Does not meet minimum technical requirements |

### 2. Experience (Weight: 25%)
| Score | Description |
|-------|-------------|
| 5 | 5+ years of directly relevant experience with leadership |
| 4 | 3–5 years of directly relevant experience |
| 3 | 2–3 years of relevant experience |
| 2 | 1–2 years of experience, partially relevant |
| 1 | Less than 1 year or no relevant experience |

### 3. Education (Weight: 15%)
| Score | Description |
|-------|-------------|
| 5 | Advanced degree (MS/PhD) from a top-tier institution |
| 4 | Bachelor's degree in CS/SE from a reputable institution |
| 3 | Bachelor's degree in a related field or bootcamp + experience |
| 2 | Some college education in a related field |
| 1 | No formal education in a relevant field |

### 4. Project Relevance (Weight: 15%)
| Score | Description |
|-------|-------------|
| 5 | Multiple production projects directly matching the role |
| 4 | Some production projects with relevant technologies |
| 3 | Personal projects or contributions showing relevant skills |
| 2 | Limited project work, mostly academic |
| 1 | No demonstrable project work |

### 5. Culture & Communication (Weight: 10%)
| Score | Description |
|-------|-------------|
| 5 | Excellent communication, clear demonstration of teamwork and initiative |
| 4 | Good communication, mentions collaboration |
| 3 | Adequate communication, basic application |
| 2 | Poor communication or unclear application |
| 1 | Very poor communication or unprofessional application |

## Decision Thresholds

| Weighted Score | Decision |
|----------------|----------|
| 3.5 and above | **Qualified** — proceed with interview scheduling |
| 2.5 – 3.49 | **Needs Human Review** — borderline candidate, escalate |
| Below 2.5 | **Not Qualified** — send rejection notification |

## Confidence Score Guidelines
- If the application provides clear, unambiguous information for scoring → high confidence (0.8–1.0)
- If some information is missing or ambiguous → medium confidence (0.5–0.79)
- If the application is very vague or contradictory → low confidence (0.3–0.49)
- Confidence below 0.7 → automatic flag for human review regardless of decision
