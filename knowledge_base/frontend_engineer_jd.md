# Frontend Engineer — Job Description

## Role Overview
We are hiring a **Frontend Engineer** to build responsive, performant, and accessible user interfaces. You will translate designs into production-ready code and work closely with backend engineers and designers.

## Responsibilities
- Build and maintain web applications using React, Next.js, or Vue.js
- Implement responsive designs with CSS/SCSS, Tailwind, or styled-components
- Integrate with REST APIs and GraphQL endpoints
- Write unit and end-to-end tests (Jest, Cypress, Playwright)
- Optimize application performance (lazy loading, code splitting, caching)
- Collaborate with UX designers to implement pixel-perfect interfaces
- Participate in code reviews and contribute to frontend architecture decisions

## Required Skills
- **Frameworks**: Proficiency in React.js or Vue.js; Next.js experience preferred
- **Languages**: Strong JavaScript/TypeScript skills
- **Styling**: Expert CSS skills; experience with Tailwind CSS or CSS-in-JS
- **State Management**: Experience with Redux, Zustand, or Context API
- **Testing**: Familiarity with Jest, React Testing Library, or Cypress
- **Version Control**: Git proficiency

## Preferred Skills
- Experience with server-side rendering (SSR) and static site generation (SSG)
- Familiarity with design systems and component libraries
- Knowledge of accessibility standards (WCAG 2.1)
- Experience with performance monitoring tools (Lighthouse, Web Vitals)
- Mobile development experience (React Native) is a plus

## Minimum Qualifications
- Bachelor's degree in Computer Science or equivalent practical experience
- 2+ years of professional frontend development experience
- Portfolio or examples of production web applications

## Compensation
- Competitive salary based on experience
- Health insurance, flexible hours, remote-friendly
